# 宽屏模式

默认情况下，聊天界面的消息区域在桌面端限制为 720px 宽度，以保证长文本的可读性。如果你希望在更大的屏幕上充分利用空间，可以切换到宽屏模式。

## 切换方式

在聊天界面右上角，点击宽屏切换图标即可：

- **切换宽屏**（↔️ 图标）：消息区域撑满屏幕宽度
- **切换窄屏**（→← 图标）：恢复默认的 720px 宽度

切换状态会自动保存在浏览器中，下次访问时保持上次的选择。

## 移动端

宽屏切换功能仅在桌面端可用，移动端默认占满全屏，不显示此按钮。
