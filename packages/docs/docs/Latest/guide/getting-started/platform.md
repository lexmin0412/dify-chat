# 开始使用

Dify App Hub 的管理后台提供了以下能力：

- 应用配置的增删改查（支持持久化存储）
- 应用配置 API，可供客户端统一拉取应用列表和配置
- Dify API 的代理服务，客户端通过 app_id 与 Platform 交互，无需直接暴露 Dify API Key

## 应用管理

使用管理员账号登录后，进入「应用管理」页面，可以看到已配置的应用列表：

![应用管理列表](/guide__platform_app_list.png)

### 新增应用

点击「新增」按钮，填写 API Base 和 API Secret，选择应用类型后确认：

![新增应用配置](/guide__platform_add_app.png)

保存成功后，应用会出现在列表中。你可以通过操作栏的按钮管理每个应用：

- **用户端** — 直接跳转到该应用的聊天界面，方便测试和调试
- **编辑** — 修改应用配置
- **同步应用信息** — 从 Dify API 拉取最新的应用名称、描述等信息
- **标注管理** — 管理该应用的标注记录
- **删除** — 移除应用配置

### 用户端入口

在应用列表中点击「用户端」按钮，会打开该应用的聊天界面：

![用户端聊天界面](/guide__platform_user_chat.png)

管理端和应用端共享同一套代码和部署，无需额外配置即可在两者之间切换。
